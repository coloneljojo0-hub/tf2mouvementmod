# TF2 Mouvement Mod

A **Team Fortress 2 movement modification** built on the **Source SDK 2013** (TF2 branch).  
This is a **work-in-progress / development build** — very unfinished, expect bugs, missing features, and frequent changes.

## About

This mod aims to enhance / overhaul movement mechanics in TF2.
and add wave-like gameplay for endless fun (and a lot of stuff)

**Current Status**: VERY Early development — not ready for general use.
